# KIDDYKODE Website

This project is a static website for KiddyKode, an initiative empowering young minds through code.

## Structure
- `index.html`: Main single-page application.
- `css/style.css`: Main stylesheet (refactored).
- `fonts/`: Custom web fonts.
- `images/`: Website assets.
- `_archive/`: Unused legacy files from the original template.

## Recent Changes
- Refactored `index.html` to use semantic HTML5 tags (`header`, `main`, `footer`).
- Removed legacy "Coffee Website Template" CSS and simplified the styling architecture.
- Archived unused pages (`about.html`, `menu.html`, etc.) for cleanup.
